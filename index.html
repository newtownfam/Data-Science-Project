Kaggle Dataset Analysis

By Mining for the Minors

About

For our project, we used a Kaggle dataset of Kaggle datasets which include attributes like dataset size, topic count, license, competition count, type and a boolean value for whether or not the dataset is featured. We used these values to create a decision tree that helps a user posting is dataset see whether or not their dataset will be featured. In particular, we encoded the dataset size, topic count, and competition count to help determine this status. Having a dataset featured determines its success as it will be easily accessible to all users, so being able to predict this using certain attributes will play a big role in helping a user determine the success of their dataset.

Factors

The factors we used in the prediction of kaggle dataset success are dataset size, topic count, and competition count. We chose these values because the user has control over them and therefore can manipulate them in order to achieve featured status.


Featured
Featured datasets appear on the main dataset page on the Kaggle webiste which may lead to more views/downloads/uses.


Dataset Size
The dataset size is simply the amount of rows by columns in a table.


Topic Count
The topic count is the amount of topics a dataset is linked to. Topics are specific keywords and phrases that pertain to the given dataset.


Competition Count
The amount of competitions a dataset belongs to. Datasets can be associated with Kaggle competitions and may encourage users to view/download/use the dataset.
